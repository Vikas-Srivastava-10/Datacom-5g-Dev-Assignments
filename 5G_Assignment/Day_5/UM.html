RLC UM Overview
Unacknowledged Mode (UM) allows data loss but provides lower latency than AM.

Used for: VoIP, Video Streaming, Real-time sensing.

Key Difference: If a packet is lost in the air interface, RLC does NOT retransmit it. The receiver detects the gap and moves on.
