5G RAN INFRASTRUCTURE

Complete End-to-End O-RAN DeploymentPRO v3.0

Radio Unit (RU)

Handles RF transmission/reception and Low-PHY layer processing. Mounted close to antennas for minimal signal loss. Supports mMIMO with beamforming.

• Functions: RF, ADC/DAC, FFT/iFFT, CP
• Interface: eCPRI over 25G SFP28
• Latency: <100μs one-way
• Power: ~800W per RU

Distributed Unit (DU)

Processes High-PHY, MAC scheduling, and RLC layers. Located at cell site for ultra-low latency requirements.

• Functions: High-PHY, MAC, RLC
• F1 Interface: 25-100Gbps QSFP28
• Latency to RU: <250μs RTT
• Deployment: Edge/Cell Site

Centralized Unit (CU)

Split into CU-CP (Control) and CU-UP (User). Handles RRC, PDCP, SDAP. Can be virtualized in data centers.

• CU-CP: RRC, Mobility, Security
• CU-UP: PDCP, SDAP, User Data
• E1 Interface: CP ↔ UP separation
• Latency to DU: <10ms F1

5G Core (5GC)

Cloud-native SBA with AMF, SMF, UPF, AUSF, UDM, PCF, NRF for authentication and session management.

• AMF: Access & Mobility
• SMF: Session Management
• UPF: User Plane Function
• Interface: SBI (HTTP/2)
