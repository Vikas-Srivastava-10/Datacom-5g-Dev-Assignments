What is PDCP Integrity Protection?

PDCP (Packet Data Convergence Protocol) integrity protection ensures that data transmitted between User Equipment (UE) and Base Station (gNB) has not been tampered with during transmission. It uses MAC-I (Message Authentication Code for Integrity) to detect any unauthorized modifications.
