What is CRC?

Cyclic Redundancy Check (CRC) is an error-detecting code commonly used in digital networks and storage devices to detect accidental changes to raw data. It's based on polynomial division in binary arithmetic.

üí° Core Purpose: CRC adds redundancy bits (checksum) to data, allowing the receiver to detect if the data has been corrupted during transmission or storage.

Why CRC?

  • ‚úÖ High Error Detection: Can detect burst errors up to the degree of the polynomial
  • ‚ö° Efficient: Simple hardware implementation using shift registers
  • üéØ Reliable: Detects all single-bit and double-bit errors
  • üì° Widely Used: Ethernet, USB, Bluetooth, ZIP files, PNG images

How CRC Works - High Level

Sender Side

1. Take data bits and append zeros (equal to CRC size)

2. Divide by generator polynomial using XOR

3. The remainder is the CRC checksum

4. Replace appended zeros with CRC checksum

Receiver Side

1. Receive data + CRC checksum

2. Divide entire received data by same generator polynomial

3. If remainder is zero ‚Üí No errors detected ‚úì

4. If remainder is non-zero ‚Üí Error detected ‚úó

‚ö†Ô∏è Important: CRC can detect errors but cannot correct them. For error correction, use FEC codes like Hamming or Reed-Solomon.

3. Generator Polynomial

The generator polynomial G(x) defines the CRC algorithm. For CRC-8:

CRC Calculation Algorithm

Input:
‚Ä¢ Data D(x) = d[n-1]...d[1]d[0] (n bits)
‚Ä¢ Generator G(x) (degree r, so r+1 bits)

Process:
1. Append r zero bits to data ‚Üí D(x)¬∑x^r
2. Divide D(x)¬∑x^r by G(x) using modulo-2 division
3. Remainder R(x) is the CRC (r bits)
4. Transmitted data = D(x)¬∑x^r + R(x)

Output:
‚Ä¢ Codeword C(x) = Data + CRC

Mathematical Properties

Detection Capabilities

CRC with degree r can detect:

‚Ä¢ All single-bit errors
‚Ä¢ All double-bit errors
‚Ä¢ All odd number of bit errors (if G(x) has even parity)
‚Ä¢ All burst errors of length ‚â§ r
‚Ä¢ Most burst errors of length > r (probability 1-2^(-r))

Verification at Receiver

Received data: C(x) = Data + CRC

Verification:

  ‚Üí No errors detected ‚úì
  ‚Üí C(x) is divisible by G(x)

If remainder ‚â† 0:
  ‚Üí Error detected ‚úó

‚öôÔ∏è Section 3: CRC Encoding & Checking Workflow

Complete CRC Process

CRC Encoding (Sender)

Step 1: Prepare Data

Original data bits: D = [d‚ÇÄ, d‚ÇÅ, d‚ÇÇ, ..., d_{n-1}]

Example: D = 11010011 (8 bits)

Step 2: Append Zeros

Append r zeros where r = degree of generator polynomial

For CRC-8: Append 8 zeros ‚Üí 1101001100000000

Step 3: Polynomial Division

Divide augmented data by generator polynomial using XOR operations

Perform modulo-2 division (XOR-based, no carries/borrows)

Step 4: Extract Remainder

The remainder from division is the CRC checksum

CRC Checksum: R = r-bit remainder

Step 5: Form Codeword

Replace appended zeros with CRC checksum

Transmitted: 11010011RRRRRRRR

CRC Checking (Receiver)

Step 1: Receive Data

Receive transmitted codeword (data + CRC)

Received: C = Data || CRC

Step 2: Polynomial Division

Divide entire received codeword by generator polynomial

No need to append zeros - CRC already included

Step 3: Check Remainder

Examine the remainder from division

Remainder = 0: No errors
Remainder ‚â† 0: Error detected

Step 4: Take Action

Based on check result:

‚úì No errors: Accept data, strip CRC, deliver to upper layer
‚úó Error detected: Discard data, request retransmission (ARQ)

Error Detection Examples

‚úì Single-Bit Error: CRC will always detect any single flipped bit
‚úì Burst Error: CRC detects all burst errors up to length r (polynomial degree)
‚ö†Ô∏è Undetectable Errors: Rare cases where multiple errors create a valid codeword (very low probability: ~2^(-r))

Error Scenarios

Error Type Detection Explanation
Single Bit 100% Always detected by any CRC
Two Bits 100% Detected if G(x) has 3+ terms
Odd # Bits 100% If G(x) contains (x+1) factor
Burst ‚â§ r 100% r = degree of polynomial
Burst > r 1-2^(-r) Very high probability

Basic CRC Calculation 
// Basic CRC implementation using polynomial division

#include <cstdint>
#include <vector>

// CRC calculation using bit-by-bit division
uint32_t calculateCRC(
    const uint8_t* data,        // Input data
    size_t length,               // Data length in bytes
    uint32_t polynomial,         // Generator polynomial
    uint8_t width                // CRC width in bits
) {
    uint32_t crc = 0;
    uint32_t msb_mask = (1U << (width - 1));
    uint32_t crc_mask = (1U << width) - 1;
    
    // Process each byte
    for (size_t i = 0; i < length; i++) {
        // XOR byte into CRC
        crc ^= (data[i] << (width - 8));
        
        // Process each bit
        for (int j = 0; j < 8; j++) {
            if (crc & msb_mask) {
                // MSB is 1, XOR with polynomial
                crc = (crc << 1) ^ polynomial;
            } else {
                // MSB is 0, just shift
                crc = crc << 1;
            }
        }
    }
    
    return crc & crc_mask;
}

// CRC-8 example
uint8_t crc8(const uint8_t* data, size_t length) {
    // CRC-8: x^8 + x^2 + x + 1 = 0x107
    const uint32_t polynomial = 0x107;
    return (uint8_t)calculateCRC(data, length, polynomial, 8);
}

// CRC-16 example (CCITT)
uint16_t crc16(const uint8_t* data, size_t length) {
    // CRC-16-CCITT: x^16 + x^12 + x^5 + 1 = 0x11021
    const uint32_t polynomial = 0x11021;
    return (uint16_t)calculateCRC(data, length, polynomial, 16);
}

// Verify CRC at receiver
bool verifyCRC(
    const uint8_t* data,        // Received data + CRC
    size_t totalLength,         // Total length including CRC
    uint32_t polynomial,
    uint8_t width
) {
    // Calculate CRC over entire received data
    uint32_t remainder = calculateCRC(data, totalLength, polynomial, width);
    
    // If remainder is 0, no errors detected
    return (remainder == 0);
}

// Example usage
void exampleUsage() {
    uint8_t data[] = {0xAB, 0xCD, 0xEF, 0x12};
    size_t dataLen = sizeof(data);
    
    // Calculate CRC-8
    uint8_t crcValue = crc8(data, dataLen);
    
    // Append CRC to data for transmission
    uint8_t transmitted[5];
    memcpy(transmitted, data, dataLen);
    transmitted[dataLen] = crcValue;
    
    // Receiver verifies
    bool valid = verifyCRC(transmitted, dataLen + 1, 0x107, 8);
    if (valid) {
        printf("‚úì CRC check passed!\n");
    } else {
        printf("‚úó Error detected!\n");
    }
}
Table-Driven CRC (Optimized) 
// Fast CRC using lookup table (256 entries)

class CRCCalculator {
private:
    uint32_t table[256];
    uint32_t polynomial;
    uint8_t width;
    uint32_t msb_mask;
    uint32_t crc_mask;

    // Build lookup table
    void buildTable() {
        for (uint32_t i = 0; i < 256; i++) {
            uint32_t crc = i << (width - 8);
            
            for (int j = 0; j < 8; j++) {
                if (crc & msb_mask) {
                    crc = (crc << 1) ^ polynomial;
                } else {
                    crc = crc << 1;
                }
            }
            
            table[i] = crc & crc_mask;
        }
    }

public:
    CRCCalculator(uint32_t poly, uint8_t w) 
        : polynomial(poly), width(w) {
        msb_mask = (1U << (width - 1));
        crc_mask = (1U << width) - 1;
        buildTable();
    }

    // Fast CRC calculation using table
    uint32_t calculate(const uint8_t* data, size_t length) {
        uint32_t crc = 0;
        
        for (size_t i = 0; i < length; i++) {
            // Table lookup - much faster than bit-by-bit
            uint8_t index = ((crc >> (width - 8)) ^ data[i]) & 0xFF;
            crc = ((crc << 8) ^ table[index]) & crc_mask;
        }
        
        return crc;
    }

    // Verify received data
    bool verify(const uint8_t* data, size_t length) {
        return (calculate(data, length) == 0);
    }
};

// Performance comparison
void performanceTest() {
    const size_t TEST_SIZE = 1000000;  // 1 MB
    uint8_t* testData = new uint8_t[TEST_SIZE];
    
    // Fill with random data
    for (size_t i = 0; i < TEST_SIZE; i++) {
        testData[i] = rand() & 0xFF;
    }
    
    // Create calculator
    CRCCalculator crc32(0x104C11DB7, 32);  // CRC-32
    
    // Calculate CRC
    auto start = std::chrono::high_resolution_clock::now();
    uint32_t result = crc32.calculate(testData, TEST_SIZE);
    auto end = std::chrono::high_resolution_clock::now();
    
    auto duration = std::chrono::duration_cast<std::chrono::microseconds>(end - start);
    
    printf("Table-driven CRC: %lu Œºs\n", duration.count());
    printf("Throughput: %.2f MB/s\n", 
           TEST_SIZE / (double)duration.count());
    
    delete[] testData;
}
Standard CRC Implementations 
// Common CRC standards with proper parameters

class StandardCRCs {
public:
    // CRC-8 (used in 1-Wire, SMBus)
    static uint8_t CRC8(const uint8_t* data, size_t length) {
        CRCCalculator calc(0x107, 8);  // x^8 + x^2 + x + 1
        return (uint8_t)calc.calculate(data, length);
    }

    // CRC-16-CCITT (X.25, Bluetooth, SD)
    static uint16_t CRC16_CCITT(const uint8_t* data, size_t length) {
        CRCCalculator calc(0x11021, 16);  // x^16 + x^12 + x^5 + 1
        return (uint16_t)calc.calculate(data, length);
    }

    // CRC-16-MODBUS
    static uint16_t CRC16_MODBUS(const uint8_t* data, size_t length) {
        CRCCalculator calc(0x18005, 16);  // x^16 + x^15 + x^2 + 1
        return (uint16_t)calc.calculate(data, length);
    }

    // CRC-24 (5G NR, LTE)
    static uint32_t CRC24(const uint8_t* data, size_t length) {
        // CRC24A: x^24 + x^23 + x^6 + x^5 + x + 1
        CRCCalculator calc(0x1864CFB, 24);
        return calc.calculate(data, length) & 0xFFFFFF;
    }

    // CRC-32 (Ethernet, ZIP, PNG)
    static uint32_t CRC32(const uint8_t* data, size_t length) {
        // x^32 + x^26 + x^23 + x^22 + x^16 + x^12 + x^11 + 
        // x^10 + x^8 + x^7 + x^5 + x^4 + x^2 + x + 1
        CRCCalculator calc(0x104C11DB7, 32);
        return calc.calculate(data, length);
    }
};

// Usage examples
void demonstrateStandardCRCs() {
    const char* message = "Hello, World!";
    const uint8_t* data = (const uint8_t*)message;
    size_t len = strlen(message);
    
    printf("Message: %s\n", message);
    printf("CRC-8:  0x%02X\n", StandardCRCs::CRC8(data, len));
    printf("CRC-16: 0x%04X\n", StandardCRCs::CRC16_CCITT(data, len));
    printf("CRC-24: 0x%06X\n", StandardCRCs::CRC24(data, len));
    printf("CRC-32: 0x%08X\n", StandardCRCs::CRC32(data, len));
}
