Scrambling in 5G NR Physical Layer

Physical Layer Processing Step

Concept Overview

What is Scrambling?

  • Purpose: Scrambling randomizes the bit sequence before modulation to improve signal quality and reduce interference.
  • Mechanism: Uses a pseudo-random sequence generator to XOR with input bits.
  • Benefits: Prevents long sequences of identical bits, reduces peak-to-average power ratio (PAPR), and improves spectral efficiency.
  • Result: Enhanced signal robustness and better channel utilization.

Key Characteristics

After Rate Matching

Applied after LDPC encoding and rate matching, before modulation

Pseudo-Random Sequence

Generated using Gold sequence or similar PRNG algorithms

Bitwise XOR Operation

Each bit is XORed with corresponding pseudo-random bit

User-Specific Sequence

Scrambling sequence depends on C-RNTI and cell ID

Scrambling Pseudo Code

// Scrambling Algorithm
Input: bitSequence[0…N-1]

// Generate pseudo-random sequence
Generate pseudoRandomSequence[0…N-1]

// XOR operation for scrambling
for i from 0 to N-1:
    scrambledBits[i] = bitSequence[i] XOR pseudoRandomSequence[i]

Output: scrambledBits[0…N-1]

Input–Output Example

Input Bits

1 0 1 1 0 0 1 0

Pseudo-Random Sequence

0 1 1 0 1 0 0 1

Scrambled Output

1 1 0 1 1 0 1 1
Scrambling = Input Bits ⊕ Pseudo-Random Sequence

Modulation in 5G NR Physical Layer

Mapping Bits to Modulation Symbols

Concept Overview

What is Modulation?

  • Purpose: Modulation converts scrambled bits into complex symbols that can be transmitted over the wireless channel.
  • Mechanism: Groups of bits are mapped to symbols using amplitude and phase variations.
  • Benefits: Enables efficient transmission, adaptive to channel conditions, supports variable data rates.
  • Trade-off: Higher-order modulation increases data rate but requires better signal quality (higher SNR).

Key Characteristics

After Scrambling

Applied to scrambled bits to create complex symbols

Bit-to-Symbol Mapping

Groups of bits mapped to constellation points

Supports Multiple Schemes

QPSK, 16-QAM, 64-QAM, 256-QAM

Adaptive to Channel

Modulation scheme changes based on channel quality

Modulation Schemes

QPSK

2 bits per symbol

Quadrature Phase Shift Keying - Most robust modulation scheme, used in poor channel conditions. Low data rate but high reliability.

16-QAM

4 bits per symbol

16 Quadrature Amplitude Modulation - Balanced performance and speed. Good compromise between throughput and reliability.

64-QAM

6 bits per symbol

64 Quadrature Amplitude Modulation - Higher data rate with moderate channel quality requirements. Commonly used in good conditions.

256-QAM

8 bits per symbol

256 Quadrature Amplitude Modulation - Very high data rate requiring strong signal. Used in excellent channel conditions.

Modulation Pseudo Code

// Modulation Algorithm
Input: scrambledBits[0…N-1]
Choose modulationScheme

// Group bits based on modulation scheme
if modulationScheme == QPSK:
    bitsPerSymbol = 2
else if modulationScheme == 16QAM:
    bitsPerSymbol = 4
else if modulationScheme == 64QAM:
    bitsPerSymbol = 6
else if modulationScheme == 256QAM:
    bitsPerSymbol = 8

// Map bit groups to complex symbols
for each group of bitsPerSymbol bits:
    complexSymbol = mapToConstellation(bitGroup)
    modulationSymbols.append(complexSymbol)

Output: modulationSymbols[]

Input–Output Example (QPSK)

Input Bits

00 01 11 10

Mapped Symbols

00→(1+j)
01→(-1+j)
11→(-1-j)

Modulation = Bit Groups → Complex Symbols

Layer Mapping in 5G NR Physical Layer

Distributing Symbols Across MIMO Layers

Concept Overview

What is Layer Mapping?

  • Purpose: Layer mapping distributes modulation symbols across multiple transmission layers for MIMO (Multiple-Input Multiple-Output) systems.
  • Mechanism: Each layer represents an independent data stream that can be transmitted simultaneously.
  • Benefits: Enables spatial multiplexing, increases throughput, improves spectral efficiency.
  • Flexibility: Supports 1 to 8 layers depending on antenna configuration and channel conditions.

Key Characteristics

After Modulation

Applied to modulated symbols to create parallel streams

Supports 1–8 Layers

Number of layers depends on MIMO configuration

Based on MIMO Rank

Rank indicates number of independent spatial streams

Parallel Data Streams

Multiple layers transmitted simultaneously

Layer Mapping Process

Modulation Symbols
↓
Layer Mapper
↓
Layer 1
Layer 2
Layer 3
Layer N

Symbols are distributed across layers depending on the MIMO rank

Layer Mapping Pseudo Code

// Layer Mapping Algorithm
Input: modulationSymbols[0…N-1]
Number of layers = L

// Initialize layer arrays
for i from 0 to L-1:
    layers[i] = []

// Distribute symbols across layers (round-robin)
for i from 0 to N-1:
    layerIndex = i mod L
    layers[layerIndex].append(modulationSymbols[i])

Output: layers[0…L-1]
