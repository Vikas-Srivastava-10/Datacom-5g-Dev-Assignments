Architecture Overview

The Network Slice Selection Function (NSSF) is a critical component of the 5G Service-Based Architecture (SBA). Its primary role is to select the set of Network Slice instances serving the User Equipment (UE).

Key Interactions:

  • N22 Interface: The connection between AMF and NSSF. AMF sends a request containing the Requested NSSAI to NSSF.
  • Nnssf Interface: A service-based interface where NSSF provides its selection and availability services to other authorized NFs.
  • Slice Selection Logic: NSSF uses Subscribed S-NSSAIs, local configuration, and Tracking Area Information (TAI) to provide the 'Allowed NSSAI' back to the AMF.
