Architecture Theory: Session Management Function (SMF)

In the 5G Service-Based Architecture (SBA), the SMF is the central control plane element responsible for managing the lifecycle of a PDU (Protocol Data Unit) session.

1. IP Address Management

The SMF acts as a DHCP server, allocating and managing IP addresses for every User Equipment (UE) connecting to the network.

2. UPF Selection & Control

It selects the best User Plane Function (UPF) for the UE based on location and traffic requirements, then provides routing rules via the N4 interface.

3. QoS Policy Enforcement

By communicating with the PCF, the SMF ensures that a gaming session gets low latency while a background download gets standard speed.

4. Signal Flow Logic

Signals move from UE → AMF → SMF. The SMF then "wires" the connection by telling the UPF how to route data to the DN (Internet).
